# Security Policy

## Reporting a Vulnerability

WealthSync handles financial data — security is our top priority.

If you discover a security vulnerability, please **do not open a public issue**. Instead, email the details to the maintainers directly.

### What to include

- Type of vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### Response

- You will receive an acknowledgment within 48 hours
- A fix will be prioritized based on severity
- You will be credited for the discovery (unless you prefer to remain anonymous)

## Best Practices

- Always enable biometric lock in the app
- Keep the app updated to the latest version
- Report suspicious behavior immediately

## Supported Versions

| Version | Supported |
|---|---|
| Latest release | ✅ |
| Older releases | ❌ |
