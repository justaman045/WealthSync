## Data Safety (Play Console → App content → Data safety)

### Data collected and shared with app

| Data type | Collected | Shared | Optional |
|-----------|-----------|--------|----------|
| Email address | Yes | No | No — required for login |
| Financial transactions | Yes | No | No — core feature |
| SMS messages | Yes | No | Yes — Pro feature, permission required |
| Biometric data | No — processed by OS only | — | — |

### Data handling

- Financial data is stored in Firebase Firestore under per-user security rules.
- SMS data is processed on-device; only extracted fields (amount, merchant, date) are stored in the user's own Firestore document.
- No data is sold or shared with third parties for advertising.
- Users can delete all data from Settings → Data Management → Delete Account.
