## The Problem

Most personal finance apps are either:
- Too basic and lack real wealth tracking
- Invasive with poor privacy
- Ugly and hard to use

**WealthSync** was built to solve this by combining beautiful design, strong privacy, AI automation, and deep wealth management in one offline-first app.

---

## Key Features

| Feature                    | Description |
|---------------------------|-------------|
| **SMS Auto-Parsing**      | On-device ML reads bank SMS and auto-categorizes transactions |
| **24 Asset Types**        | Track stocks, mutual funds, real estate, crypto, gold, PF, NPS, etc. |
| **Biometric Security**    | App lock with fingerprint/face ID |
| **Offline-First**         | Works completely offline with local storage |
| **UPI Integration**       | Send money directly from the app |
| **Glassmorphic UI**       | Modern, beautiful design with smooth animations |
| **AI Insights**           | Smart spending analysis and savings suggestions |
| **Multi-Account Support** | Track multiple bank accounts and wallets |

## Tech Stack

| Layer            | Technology                          |
|------------------|-------------------------------------|
| **Framework**    | Flutter 3.35 (Dart)                 |
| **State Mgmt**   | GetX / Riverpod                     |
| **Database**     | Firebase Firestore + Hive (local)   |
| **Auth**         | Firebase Authentication + Biometrics|
| **AI / ML**      | On-device ML for SMS parsing        |
| **Payments**     | UPI + Razorpay (future)             |
| **Theming**      | Glassmorphism + Dark Mode           |
